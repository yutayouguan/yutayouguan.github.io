禁用发光\毛玻璃特效和卸载

发光效果始于一个笑话，从未打算用于长期编码会话。如果要将其关闭，可以随时通过使用Ctrl + Shift + P或打开命令选项板来禁用它，Shift + ⌘ + P然后选择“Disable custom CSS”。

如果您决定完全更改为其他主题，则还需要以这种方式禁用自定义css。您可能还希望从您的文件中删除文件路径settings.json。
